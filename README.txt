In this Assignment, I built a basic messaging app with a simple GUI (in a4.py) and a server connection to allow users to message each other online. The ds_protocol.py and ds_messenger.py allow for a client-server connection and communication. The Profile.py allows users to be identified by a dsuserver, username, and password, message other users, and add other users. There is also a testing module for both ds_protocol.py and ds_messenger.py to ensure server communication and connection work properly.

Once you run the program (from a4.py), a messaging GUI will pop up. At the top left of the screen, to start messaging people, click on 'Configure DS Server'. There, you can enter the DS Server Address, Username, and Password of your profile. You can then go back to Settings, click Add Contact, and enter the username of the profile you want to message. In the bottom right box, type the message you want to send, and then click the green 'send' button at the very bottom right corner of the GUI. That will print your message to the chat box, and any messages sent to you will also appear there. Additionally, any user who messages you or that you have added will appear in the contacts list on the left side of the GUI.
